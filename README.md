## Approach Taken
- Broke the landing page into reusable React components.
- Used Next.js App Router structure.
- Styled the complete UI using pure CSS instead of Tailwind.
- Focused on responsiveness, clean layout, and section-based navigation.
- Recreated major sections of the reference site while keeping code modular.
